# The Real Truth

## Finality Before Inquiry

A public documentary case file examining how an open clinical question became a settled administrative fact.

This site is not a litigation document.

It is an archive of sequence, language, and record behaviour.

The governing question:

> If no one could know, no one could certify as if they knew.

## Structure

- `index.html` — public landing page
- `style.css` — visual system
- future `/essays` — long-form narrative analysis
- future `/documents` — primary record guides

## Public Thesis

The case does not require the reader to believe the patient over the documents.

It requires the reader to believe the documents over the institution's later use of them.
